# Snake-water---gun-game
Snake Water Gun Game
A Python-based implementation of the classic Snake Water Gun game where the user competes against the computer.
Project Overview
This game allows the user to choose Snake, Water, or Gun while the computer generates a random choice. The winner is determined using predefined game rules.
Features
User vs Computer gameplay
Random computer choice generation
Winner determination using conditional logic
Easy-to-use command-line interface
Beginner-friendly Python project
Technologies Used
Python
Random Module
How to Run
Download or clone the repository.
Open the project in Python.
Run the snakewatergun.py file.
Enter your choice (Snake, Water, or Gun).
View the game result.
Game Rules
Snake drinks Water → Snake wins
Water douses Gun → Water wins
Gun kills Snake → Gun wins
Same choices result in a draw
