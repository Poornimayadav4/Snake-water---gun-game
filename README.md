# Snake-Water-Gun Game 🐍💧🔫

A simple Snake Water Gun game built using Python where the player competes against the computer.

## Features
- User vs Computer gameplay
- Random computer choices
- Win, lose, and draw conditions
- Interactive command-line interface
- Easy-to-understand game logic

## Technologies Used
- Python
- Random Module
- Conditional Statements
- Dictionaries
- User Input Handling

## Learning Outcomes
- Working with the random module
- Using dictionaries in Python
- Implementing game logic with if-elif-else
- Handling user input
- Building interactive console applications

## How to Run
1. Open the project in Python.
2. Run the program.
3. Enter:
   - `s` for Snake
   - `w` for Water
   - `g` for Gun
4. The computer will make a random choice.
5. The winner will be displayed.

## Game Rules
- Snake drinks Water → Snake wins
- Water douses Gun → Water wins
- Gun kills Snake → Gun wins
- Same choices result in a draw

## Sample Gameplay
